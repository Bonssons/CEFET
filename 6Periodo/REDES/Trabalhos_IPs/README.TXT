O que o programa faz:
	A partir de um número de máscara válido no primeiro octante, o programa retorna todas as subredes possíveis neste intervalo, evidenciando os IPs de rede, broadcast, inicial e final.

	O arquivo input.in possui uma sequência de todas as entradas válidas para o programa.


Compilar:
gcc ips.c

Executar:
./a.out
ou 
./a.out < input.in
